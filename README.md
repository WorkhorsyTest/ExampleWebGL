# Example WebGL

# Checkout code
```
git clone https://github.com/workhorsy/ExampleWebGL
cd ExampleWebGL
```

# Run in browser
```
py -3 -u -m http.server 9991
py -2 -u -m SimpleHTTPServer 9991
```
